# 杉ヶ平キャンプ場 予約システム 試作版

2026年6月20日 NPO法人 大長谷村づくり協議会 理事会向けデモ。

## 概要

富山県富山市八尾町・21世紀の森内「杉ヶ平キャンプ場」のオンライン予約システム試作版です。

- **お客様向け**：トップページ／予約フォーム／予約確認・変更
- **スタッフ用（管理画面 `/admin`）**：ホーム／今日の受付／段取り／予約一覧／顧客一覧

本リポジトリは設計検証・理事会デモ用の試作版です。実データは含まれません（すべてダミーデータ）。

## 技術スタック

- Next.js 16.2.4（App Router、Turbopack）
- React 19.2.4
- TypeScript
- Tailwind CSS

## 開発

```bash
npm install
npm run dev
```

http://localhost:3000 で起動します。
